Details

A webpage that detect whether the input news article text is true or false by using Machine Learning algorithm.

The home page provides an input where the user can copy-paste any text, which is then pre-processed using technologies such as tokenization, stemming, stop-word removal, and spell-correction.

After pre-processing, the text is tested against 1000 trained news article which are marked fake or real and an output is provided.

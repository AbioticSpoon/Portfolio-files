Details

A node app that can create, read, edit and destroy notes as per the wish of the user. This app follows the restful conventions and rules.

The home page displays all the notes that are inserted by the user with two buttons, one to read the complete markdown of the notes and one to delete the notes. On the read-more page, the user can edit his/her notes.

The front-end is styled using bootstrap.

Technologies used

  • HTML
  • BOOTSTRAP
  • NODE.JS
  • MONGO-DB

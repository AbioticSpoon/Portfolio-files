Details

This project is a simple website that take user inputs such as Personal Details, Education details, Experience details and Skills and generates a pdf file in form of a Resume.

The website contains of 2 total comoponents, a hero comoponents, a form comoponents with 4 total sub-components which include forms to accept various user details neccessary for the resume.

After accepting the user details, the form is converted to pdf format using a react-pdf npm module, This feature is still gimmicky and I am working towards improving it.

Technologies used

  • React.JS
  • Bootstrap 4
